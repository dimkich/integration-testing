### Модули и возможности

* **[Аннотация JsonMapAsEntries](JsonMapAsEntries.md)** — Расширенный маппинг коллекций для XML/JSON: поддержка
  ключей-атрибутов и оберток для записей.
* **[Механизм ResettableIterator](ResettableIterator.md)** — Многоразовые итераторы, предотвращающие потерю данных при
  логировании или повторных проверках.
* **[Система Wait-Completion](wait-completion.md)** — Решение проблем асинхронности в интеграционных тестах (реактивные
  стеки, шины событий).
